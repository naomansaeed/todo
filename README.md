# Todo List API (Node.js)

## 🎯 Goal
Learn Node.js fundamentals by building a file-backed CRUD API before moving to Express, MongoDB, and React.

## 📦 Tech Stack (Current Phase)
- Pure Node.js (no frameworks)
- `fs` module + JSON for data storage
- Testing via `curl` or Postman

## 🚀 Setup & Run
```bash
node index.js
# Server runs at http://localhost:3000
